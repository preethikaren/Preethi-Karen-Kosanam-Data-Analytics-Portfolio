Apple Stock Price Forecasting (P-559)

>Objective
Forecast Apple Inc. (AAPL) stock prices for the next 30 days to help investors, traders, and analysts make data-driven buy/sell decisions.

>Tools & Technologies
    Python, Pandas, NumPy, Statsmodels, Streamlit

>Project Overview
The project uses historical Apple stock prices along with economic, financial, and sentiment indicators to identify patterns and predict future prices. Multiple time-series and machine learning models (VAR, ARIMA, Random Forest) were evaluated to achieve the best forecasting accuracy.

>Project Structure
- `code/` : Streamlit app and Jupyter notebook  
- `dataset/` : Historical stock data  
- `outputs/` : Dashboard screenshots and presentation  

>Outcome
Developed an interactive Streamlit dashboard that allows real-time visualization of predicted stock prices, helping users make informed investment decisions.
